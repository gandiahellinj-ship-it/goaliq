Add animated SVG exercise illustrations to the Workouts page.

For each exercise card, add a simple animated SVG figure on the right side showing the movement. The figure should be a minimal stick-figure or geometric human silhouette in lime green (#AAFF45) on the dark background.

## Implementation

Create a set of SVG animations for these common exercise categories:

1. SQUAT (matches: squat, lunge, leg press, step up)
- Figure bending knees up and down in a squat motion
- CSS animation: loop, 2s duration

2. PUSH (matches: push-up, chest press, shoulder press, overhead press, dip)
- Figure doing a push/press motion, arms extending and retracting
- CSS animation: loop, 2s duration

3. PULL (matches: row, pull-up, lat pulldown, curl, face pull)
- Figure doing a pulling motion, arms drawing in
- CSS animation: loop, 2s duration

4. HINGE (matches: deadlift, hip thrust, good morning, romanian)
- Figure hinging at the hips, torso going forward and back
- CSS animation: loop, 2s duration

5. CORE (matches: plank, crunch, sit-up, ab, core, twist)
- Figure doing a crunch motion
- CSS animation: loop, 2s duration

6. CARDIO (matches: run, jump, burpee, sprint, cycling, bike)
- Figure doing a running/jumping motion
- CSS animation: loop, 1.5s duration

7. DEFAULT (anything else)
- Simple figure doing a generic movement
- CSS animation: loop, 2s duration

## Matching logic
Write a function: getExerciseAnimation(exerciseName: string) that:
- Lowercases the exercise name
- Checks for keywords from the categories above
- Returns the matching SVG component
- Falls back to DEFAULT if no match

## Layout per exercise card
- Left side (existing): exercise number, name, sets x reps, rest time
- Right side (new): animated SVG illustration, width 80px height 80px
- SVG figures: stroke color #AAFF45, stroke-width 2.5, no fill, clean minimal lines
- The animation should loop continuously and smoothly

## Style
- Keep the dark card background
- The SVG sits on the right side of each exercise row
- On mobile: show the SVG above the exercise details or hide it on very small screens (under 380px)
- The figures should look clean and intentional, not rough

## SVG figure style guide
Use simple geometric shapes to represent a human:
- Head: small circle
- Body: single line (torso)
- Arms: two lines from shoulders
- Legs: two lines from hips
All connected, proportional, minimal. Think icon-style human figure, not detailed anatomy.

Do not use any external libraries or APIs. Pure SVG + CSS animations only.
