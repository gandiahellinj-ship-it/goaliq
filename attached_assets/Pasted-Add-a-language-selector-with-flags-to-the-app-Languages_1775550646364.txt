Add a language selector with flags to the app.

## Languages supported
- English (default) — flag: 🇬🇧
- Spanish (Spain) — flag: 🇪🇸

## UI placement
- Top right corner of the app, visible on every page
- Show the current language flag as a small clickable button (28x28px)
- On click: show a small dropdown with both flags + language name:
  🇬🇧 English
  🇪🇸 Español
- Active language has a lime green #AAFF45 border/underline
- Dropdown background: #1A1A1A, border 1px solid #2A2A2A, border-radius 8px

## Behavior
- Default language: English
- When user switches to Spanish: entire app UI changes to Spanish instantly
- Language preference saved in localStorage so it persists across sessions
- If user has no saved preference: default to English

## Implementation
Use a simple React context (LanguageContext) with a useTranslation() hook:
- t('key') returns the string in the current language
- All components use t() instead of hardcoded English strings
- Easy to add more languages later

Do not change any functionality, only add translations and the flag selector UI.
