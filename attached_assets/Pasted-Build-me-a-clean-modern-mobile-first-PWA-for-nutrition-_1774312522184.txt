Build me a clean, modern, mobile-first PWA for nutrition and training coaching.

Main goal:
Create an intuitive and friendly app that helps users manage their diet and training plan in a visual way, so they stay motivated and learn to be more independent over time.

Core features:
1. User authentication:
- Sign up
- Login
- Logout

2. Onboarding:
Create a step-by-step onboarding flow where the user enters:
- age
- sex
- height
- weight
- goal (lose fat, maintain, gain muscle)
- diet type (balanced, keto, vegan, vegetarian, high protein)
- allergies and intolerances
- foods they like
- foods they dislike
- training level (beginner, intermediate, advanced)
- training location (gym, home, outdoor)
- training days per week

3. Meal plan:
After onboarding, automatically generate a weekly meal plan based on:
- selected diet type
- user goal
- allergies and intolerances
- liked foods
- disliked foods

The weekly meal plan must be visual and easy to understand.
Each meal card should show:
- meal name
- ingredients
- simple portion idea
- a visual plate distribution using percentages
Example:
- 50% vegetables
- 30% protein
- 20% carbs

4. Ingredient replacement:
Each meal must include a button:
- "I don't like this ingredient"
When clicked, the app should replace that ingredient with another food from the same category in an intelligent way.
Example:
- rice -> potato or quinoa
- salmon -> chicken or tofu
- yogurt -> lactose-free yogurt or soy yogurt

5. Training plan:
Automatically generate a weekly training plan based on:
- user goal
- level
- training days
- training location

The weekly workout plan must show:
- day of the week
- workout type
- exercises
- sets and reps or duration
- short guidance

6. Calendar:
Create a weekly and monthly calendar view where the user can:
- see past and future planned workouts
- mark workouts as completed
- review previous weeks and months
- connect food and training adherence visually

7. Progress:
Create a progress page with:
- current weight
- target weight
- weekly adherence
- completed workouts
- simple motivational progress visuals

8. Subscription plans:
Create a pricing page with 3 plans:
- Basic
- Pro
- Premium

9. Design:
The design must be:
- modern
- clean
- visual
- friendly
- motivating
- simple
- mobile-first
Use soft colors, rounded cards, clear buttons, and a premium wellness feel.

10. Pages required:
- Landing page
- Sign up / Login
- Onboarding
- Dashboard
- Weekly meal plan
- Weekly workout plan
- Calendar
- Progress
- Pricing / Subscription

Technical requirements:
- Build it as a PWA
- Add installable app behavior
- Add a web app manifest
- Prepare offline-friendly structure
- Keep the code organized and editable
